# SnowX - 轻量级Python异步事件驱动框架

> 极简、高效、可扩展的Python异步事件驱动框架，专为插件化架构设计

## 项目简介
SnowX 是一款零外部依赖的Python异步事件驱动框架，核心聚焦于**插件化开发体验**与**运行稳定性**。

## 快速开始
### 环境要求
- Python 3.12+
- 无其他外部依赖

### 部署运行
1. 克隆代码仓库
```bash
git clone https://github.com/blueiris0031/snowx.git
cd snowx
```

2. 启动框架
- Linux/macOS：
  ```bash
  chmod +x run.sh
  ./run.sh
  ```
- Windows：
  ```
  run.cmd
  ```

3. 验证运行
框架启动后会自动加载默认插件，控制台输出类似以下内容即表示运行成功：
```
[INFO] SnowX framework starting...
[INFO] Loading core plugins...
[INFO] Plugins loaded successfully (total: X)
[INFO] SnowX framework started successfully!
```
